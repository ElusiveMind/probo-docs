Probo integrates with Bitbucket, a git repository management system created by Atlassian. Probo also supports Bitbucket Server, however it requires manual setup on our part.

Probo uses Bitbucketâ€™s build status pop-up to display the list of build steps and their statuses. The link can be found in the Overview section of the pull request detail page.

Developers who are reviewing the pull request can see whether any automated tests failed against a Probo site build.

Click the title of a build step to be directed to the Probo app, where you can find the console output for each step and a link to the sandbox site.
