Drupal Plugin

Use the Drupal plugin to make it easy to define and declare build steps. The installArgs parameter defines any necessary arguments relating to the build. The Drupal plugin is using the makeFile and the drush command drush site-install. Look at the Drush documentation for more about the site-install command. For more parameter options read more in the docs about different parameter options. *Note for this option drush site-install will drop and rebuild all the tables in the database.
